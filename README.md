# sftp-validation
Java链接SFTP验证工具

版本V0.0.1

--------------------参数说明-------------------------

--host                     :FTP地址 默认是localhost

--port                     :FTP端口

--username                 :FTP用户名

--password                 :FTP密码

--privateKey               :ssh秘钥

--strictHostKeyChecking    :是否开启强验证 yes or no

--path                     :要ls的路径
